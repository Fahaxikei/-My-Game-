This is a first-person shooter (FPS) game that I am still learning and developing, built using Unreal Engine 5.

Please note! The game is currently developed using Blueprints only, and the game language is Simplified Chinese only.

The game assets and mods are sourced from the Unreal Engine 5 Marketplace.

Learning resources are from: https://www.bilibili.com/cheese/play/ss32685?bsource=link_copy

Currently, the game has implemented: basic character movement, map construction, supply pickups, basic enemy design, fundamental character attacks, simple UI design, and core gameplay mechanics.

Future plans include implementing a weapon modification system, a third-person perspective toggle, and some special attack moves.
